How It Works?

Whatsapp Backup Reader has the goal of converting the ugly exported .txt whatsapp chats to a readable version!

You can always check video demonstration

Basically All you have to do is:

  • Export Whatsapp chat and move it to your computer.
  • Upload the text file to this tool and download the index.html file
  • Make sure you put the index.html at the same directory with other medias
  • Congrats! You just converted an ugly text file to a readable HTML page.
